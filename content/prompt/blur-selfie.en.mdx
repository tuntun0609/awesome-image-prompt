### Prompt

```text
请画一张极其平凡无奇的iPhone 自拍照，没有明确的主体或构图感，就像是随手一拍的快照。照片略带运动模糊，阳光或店内灯光不均导致轻微曝光过度。角度尴尬、构图混乱，整体呈现出一种刻意的平庸感-就像是从口袋里拿手机时不小心拍到的一张自拍。主角是邓布利多教授，晚上，在悉尼歌剧院旁
```

Characters and scenes in the prompt can be replaced

> When using the original prompt with Doubao, the phone appears in the image, so you need to add "don't show the phone in the image" to the prompt
